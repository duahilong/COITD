## 基础信息

### 项目名称

`COITD`

### 当前约束

1. 项目主要运行在 Linux 服务器上。
2. 外围控制逻辑优先使用 `python3` 与 Shell。
3. `CloudflareSpeedTest/` 为上游只读目录：仅允许读取，不在本仓库中直接修改。
4. 代码和文档应尽量围绕当前可维护的主流程组织，避免保留未完成脚本与阶段性残留文件。

### 当前主流程

1. `scripts/cfst/`：配置化运行 CFST，并提供任务控制能力。
2. `scripts/webui/`：CFST Web 控制台。
3. `scripts/cfst_ddns/`：AliDNS DDNS 推送脚本。
4. `scripts/cfst_ddns_web/`：DDNS 专用 Web 监控页。
